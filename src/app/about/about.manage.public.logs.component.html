The Manage Public Logs menu allows you to make logs public or change the settings on logs you have already made public.
The fields and buttons are:   This control allows you to specify which of you matches to show on the menu. You can specify only Private matches or matches you have already made Public or Both.   This button will display next to the item if the match is currently a Private match only. Press this button to make a Public copy of this match.   This button will display next to the item if the match there is currently a Public copy of this match. Press this button to change the settings of the Public copy.
