The origins list contains the names of the person, media, etc. where a recipe came from, such as 'Martha Setwart' or 'Jane E. Gordon'.
The Manage Origins form allows you to add/remove/change entries in your origins list.
The fields and buttons are:   This is the name for the origin. From the drop-down list, select an origin to change/remove it or select New Origin Name to add a new origin name to the list.   This icon will display next to the input field if you choose an exitsig origin from the list in the Origin Name field. Edit or delete this origin name.   This icon will display next to the input field if you choose New Origin Name from the list in the Origin Name field. Enter a name for the new origin here.   Press this button to submit your list update request. If successful, you will receive a brief notification.   Press this button to remove the selected origin from the list.   Press this button to return to name selection without changing the current selection.   If you delete an origin, any recipes in your collection that still have that origin will now show 'Removed' as the origin when displayed. You can fix that by editing the recipe and specifying a different origin.
