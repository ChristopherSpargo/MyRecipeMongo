The locations list contains the names of the locations where tournaments/matches are played, such as 'Scottsdale Ranch Park'.
The Manage Locations form allows you to add/remove/change entries in your locations list.
The fields and buttons are:   This is the name of a of a tournament/match location. From the drop-down list, select a location to change/remove it or select New Location Name to add a new location name to the list.   This icon will display next to the input field if you choose an exitsing location from the list in the Location Name field. Edit or delete this location name.   This icon will display next to the input field if you choose New Location Name from the list in the Location Name field. Enter a name for the new location here.   Press this button to submit your list update request. If successful, you will receive a brief notification.   Press this button to remove the selected location from the list.   Press this button to return to name selection without changing the current selection.
