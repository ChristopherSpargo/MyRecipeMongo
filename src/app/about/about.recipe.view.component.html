The View Statistics tab displays all point information from the selected match log in statistical form.
  This tab is organized in four sections:
Point Filter, Setup Information,
Menu and Statistics.   For a single log, this section contains the point filter activation button and the filtered points navigation control.
Press the to open the Point Filter form. If a filter has been applied and matching points are found, the navigation control will appear allowing you to traverse the list of matching points.   For a single log, this section displays general information about the match such as tournament name, date, time, match format, etc.
  When multiple logs have been selected, this section displays a count of the matches, the number of unique opponents in the data and the dates of the beginning and ending matches in the data.
  To toggle the display of this section, tap the icon on the VIEW tab at the top of the tab display.   When a single match has been selected, this section displays a menu for selecting the level of information you want to see. The stats for an individual match can be viewed at the Match, Set, Game and Point level.
  The Match level shows the players names on a button with Set buttons to the right showing the score of each set. There is also a button which will automatically switch the display to show detail information for the final point in the log.
  Selecting a set button displays a row of buttons each representing a game in the selected set or a point in the selected set in the case of a Tiebreak set.
  The statistics graphs/text change to reflect the chosen set.
  Game buttons display a check mark and a score. The check indicates the winner, red for break and green for hold. The score represents the loser's score when the game finished.
  Selecting a game button displays a row of buttons each representing a point in the selected game.
  The statistics header will now display a small ball next to the player's name that served that game and the set score when that game began.
  Graphs are not displayed at the game level, only the text for the game statistics is shown.
  Point buttons display each player's score when the point started. Also, the colored score on the button indicates who won that point, red for returner, green for server.
  Finally, selecting a point button displays an account of that point which includes all relevant information from the Log Point tab.   When multiple matches are selected, this section simply displays the name of the player and how many different opponents are in the selected matches. In this situation, statistics are displayed only for the match level.   Here is where the actual statistics are displayed.
  This section begins with a header that shows the players names on their side of the statistics display.
  In the middle of the header is a description of the information level and the score when that game/point began if appropriate.
  A check mark will display after the name of the player that won that level, red if returning and green otherwise.
  At the game and point levels, a small tennis ball will be displayed before the name of the server.
  The header is followed by four subsections:
Synopsis, Serving, Returning and Risk.
Each subsection can be displayed or hidden by tapping the
Section Name button at the top-left of the section. The text representation of the data can be displayed or hidden by tapping the
Text button at the top-right of the section.   The synopsis section gives you an overview of how each player is performing. The text display shows data as ratios or counts whereas the graph display shows data as percentages. In general, to be doing well a player's performance should be over 50% in every category on this graph.   The serving section gives you information of how well each player is serving. Information includes counts of serves in play and won, aces, double faults and Game points won. The text display shows data as ratios or counts whereas the graph display shows data as percentages. In general, to be doing well a player's performance should be over 50% in most of the percentage categories on this graph.   The returning section gives you information of how well each player is returning serve. Information includes counts of returns in play and won and Break points won. The text display shows data as ratios or counts whereas the graph display shows data as percentages. In general, to be doing well a player's performance should be near 50% in most of the categories on this graph.   The risk section gives you information of how aggressively each player is playing. Information includes counts of winners, errors forced, aces, double faults, unforced errors, points at net, total points won, average rally length and longest rally. Both the Graph and Text displays in this section show data as ratios or counts.
  In general, higher numbers of winners and/or unforced errors along with shorter rallies and possibly coming to net indicates more aggressive play whereas lower winners/unforced errors and longer rallies while staying back shows more conservative play.   Trend graphs display match statistics as a set of points for each match going horizontally across the display. The lines connecting the points show the fluctuations in performance over time from left to right. Each category displays two graphs, one for Player and one for Opponents.
  Touching a display point on the graph displays a pop-up with the statistic for that match on that date. For the Opponents graph, the pop-up also contains the name of the opponent.   Graphs are displayed only for the Match and Set levels. Also, some information shown in the Text display is not shown in the Graphs.
  Items on a graph can be excluded or included by tapping their entry in the legend.
  Touching the graph will display a pop-up with the information being displayed at that position, if any.
