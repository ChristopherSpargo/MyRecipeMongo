This tab of the Recipe Access Area is where information about a recipe is entered/edited.
The fields and buttons are:   This is the title of the recipe. Each word of the title will be capitalized automatically. Words/phrases in the title can be searched for using the Keywords field of the SEARCH tab.   Enter a description of the dish created from this recipe. The description is displayed on a search results menu.   Here you assign categories to the recipe. Categories are one of the main ways to organize recipes. You can create as many different categories as you like (see help when using the Manage Categories feature).
  Click the button to display a menu of categories. Add categories to the recipe by checking any you want to apply and click . Selected categories will appear in the categories field as labeled chips.
  To remove a selection click the on the chip with that category label.   Where the recipe originated. Select an entry from the list.   The year the recipe originated. Enter a 4-digit year.   This field is for adding any clarifying information about the recipe's origin.   Enter the list of ingredients for the recipe. Type in each ingredient followed by the enter key. The program will automatically bullet each ingredient when you hit the enter key. You can enter subtitles like 'FOR THE SAUCE:' by ending the line with a semicolon. The program will display subtitles in all caps. Blank lines (lines that start with the enter key) will be maintained.
 Words/phrases in the ingredients can be searched for using the Keywords field of the SEARCH tab.   Enter the list of instructions for the recipe. Type in each instruction followed by the enter key. The program will automatically number each instruction when you hit the enter key. You can enter subtitles like 'FOR THE SAUCE:' by ending the line with a semicolon. The program will display subtitles in all caps. Step numbers start over after a subtitle. Blank lines (lines that start with the enter key) will be maintained.   Enter any special considerations regarding ingredients or preparation of the recipe.   This field is for adding pictures to the recipe. The first picture entered will be the one displayed on a search results menu. Tap the button to browse for pictures and/or to use the camera on a mobile device.   Press this button to save the recipe to the database and update the VIEW tab.   Press this button to clear all recipe information for a NEW recipe and start over.   Press this button to cancel an EDIT of an existing recipe and return all the fields to their original values.   If you will be sharing this recipe, be sure to add your own personal category before sharing the recipe. This will make it easy for users to find recipes you have contributed.
