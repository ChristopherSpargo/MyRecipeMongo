The Add Public Match form allows you to make a Public copy of the selected match.
The fields and buttons are:   This is an optional list of users that will be allowed to view the Public copy. The list contains email addresses that are associated with the accounts of the users you want to allow access to this log. Leave this list empty to allow all users to view this log.
From the drop-down list, select an email address to change/remove or select New Authorized User to add a new email address to the list.   This button cancels making the Public Copy.   This button creates the Public copy of the match and saves the Authorized Users List if any.
