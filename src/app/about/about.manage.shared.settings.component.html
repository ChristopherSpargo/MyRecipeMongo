The Shared Recipe Settings form allows you to change the settings for a recipe you have Shared.
The fields and buttons are:   Check this box to allow removal of the Shared copy of the recipe.   This is an optional list of users that will be allowed to view the Shared copy. The list contains email addresses that are associated with the accounts of the users you want to allow access to the Shared copy this recipe. Leave this list empty to allow all users to view this recipe.
From the drop-down list, select an email address to change/remove or select New Authorized User to add a new email address to the list.   This button cancels making changes to the Shared copy settings.   This button saves changes to the Shared Copy settings.   This button removes the Shared copy of the recipe.
