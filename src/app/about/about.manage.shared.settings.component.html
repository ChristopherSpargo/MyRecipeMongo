The Public Match Settings form allows you to change the settings for a match you have made Public.
The fields and buttons are:   Check this box to allow removal of the Public Copy of the match.   This is an optional list of users that will be allowed to view the Public copy. The list contains email addresses that are associated with the accounts of the users you want to allow access to this log. Leave this list empty to allow all users to view this log.
From the drop-down list, select an email address to change/remove or select New Authorized User to add a new email address to the list.   This button cancels making changes to the Public Copy settings.   This button saves changes to the Public Copy settings.   This button removes the Public copy of the match.
