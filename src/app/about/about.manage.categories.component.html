The events list contains the names of the events players play in, such as 'Girls 18S', when they enter tournaments.
The Manage Events form allows you to add/remove/change entries in your events list.
The fields and buttons are:   This is the name of an event players can play in as part of a tournament. From the drop-down list, select an event name to change/remove or select New Event Name to add a new event name to the list.   This icon will display next to the input field if you choose an exitsing event from the list in the Event Name field. Edit or delete this event name.   This icon will display next to the input field if you choose New Event Name from the list in the Event Name field. Enter a name for the new event here.   Press this button to submit your list update request. If successful, you will receive a brief notification.   Press this button to remove the selected event from the list.   Press this button to return to name selection without changing the current selection.
