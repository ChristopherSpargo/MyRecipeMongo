Here's a little Quick Start information to help you get going with MyRecipeMongo.   MyRecipeMongo is a website where you can organize and share recipes.
  Organize recipes by categories, origin, and origin date. Retreive recipies using categories, origin, date or keywords found in the title or ingredients.
  Share recipes with everyone or just specific users.   Recipes are stored in the cloud and can be accessed at anytime from any device.   The first thing you'll need to do to start using MyRecipeMongo is create an account. Since all data is stored in the cloud, we need to know which data is yours. MyRecipeMongo accounts are simple, just an email address and a password.

  To create an account tap on the main screen.
  Then, on the Sign In form:
    - Enter an email address and password
    - Check the Create new account box
    - Enter the same password again
    - Finally, tap
  If at any time you are unsure of what to do just tap the icon in the upper left corner of the screen and a help panel, like this one, will be displayed that has information relevant to the current activity.   On smaller screens, the full command menu is available by pressing the icon at the far right of the MyRecipeMongo title bar. The title bar also contains a few quick access icons for the most commonly used features.
  On larger screens, the full menu is displayed in the title bar with dropdowns for the commands.   To add a recipe tap on the main screen (or navigate to Recipes>Add a Recipe on the main menu).
  Every recipe starts out with the NEW (Add Recipe) tab of the Recipe Access Area. This is where all of the information for the recipe is entered.
  This information includes:
    Title
    Description
    Categories
    Recipe Origin
    Origin Date
    Ingredients
    Instructions
    Recipe Notes
    Pictures
  Choices for Categoriesand Origin are maintained by you in separate lists using the List Management menu. Each function in List Management has its own help panel.   To find and view recipes, tap the icon in the main title bar (or navigate to Recipes>Recipe Search on the main menu). This will open the Recipe Access Area with the SEARCH tab active.
  Fill in any search criteria and tap . If matching recipes are found, a menu of those recipes will be displayed on the RECIPES tab. Tapping the picture or description on the menu item will display the full recipe in the VIEW tab.

  To access a different tab of the Recipe Access Area, tap its name at the top of the display.
